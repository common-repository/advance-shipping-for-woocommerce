== Short Description ==

WooCommerce Advanced Conditional Shipping is a powerful shipping management system that allows online merchants to customize their shipping rates and options based on various conditions. With this plugin, you can create and apply specific rules to ensure accurate and flexible shipping calculations.

== Installation ==

Download the .zip file.
Go to: WordPress Admin > Plugins > Add New and Upload Plugin with the file you downloaded with Choose File.
Install Now and Activate the extension.
